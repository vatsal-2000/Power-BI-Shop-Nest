ShopNest Sales Analytics Dashboard
Overview

This project analyzes ShopNest sales data using Power BI to uncover trends in revenue, customer behavior, product performance, and order fulfillment. The dashboard enables business stakeholders to monitor key metrics and make data-driven decisions.

Objectives
Analyze overall sales performance
Identify top-performing products and categories
Track customer purchasing behavior
Monitor order delivery performance
Visualize revenue and profit trends
Tools Used
Power BI
SQL
Excel/CSV
DAX
Key KPIs
Total Revenue
Total Orders
Average Order Value
Customer Count
Top Selling Products
Category-wise Revenue
Monthly Sales Trend
Delivery Performance Metrics
Dashboard Features
Interactive filters and slicers
Revenue trend analysis
Product category performance
Customer segmentation insights
Regional sales breakdown
Dynamic KPI cards
Key Insights
Identified top revenue-generating product categories.
Highlighted seasonal sales trends and peak purchasing periods.
Analyzed customer purchasing patterns to support targeted marketing strategies.
Evaluated delivery performance and fulfillment efficiency.
Project Outcome

Developed an interactive business intelligence dashboard that transforms raw sales data into actionable insights, helping stakeholders monitor performance and identify growth opportunities.
